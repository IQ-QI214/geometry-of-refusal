## 1. 执行摘要

我们进行了 12 组消融实验（2 个模型 × 2 种方法 × 3 种锥体维度），以验证 Qwen2.5-VL-7B 的隐蔽拒绝行为是由"不完整的锥体"导致（假设 A），还是由独立的安全机制导致（假设 B）。

**核心结论**：数据强有力地支持**假设 B**。即使在最强攻击方法 RDO（k=5）下，Qwen 的隐蔽拒绝率（SRR）仍保持在 60% 以上，且经裁判模型评估的真实攻击成功率（ASR）从未超过 9.4%（LLaMA-Guard）/ 7.0%（Qwen3Guard）。这表明：拒绝方向与隐蔽拒绝机制在架构上是相互独立的。

**三个意外发现**：

1. **退化代价（Degeneration Tax）**：LLaVA 表面上的关键词攻击成功率（keyword ASR）被退化输出（如令牌坍缩）夸大了 3–6 倍。其真实攻击成功率约为 10–16%，而非 54–73%。
2. **RDO 悖论**：在 Qwen 上，经梯度优化的 RDO 锥体在关键词绕过方面反而不如简单的 DIM PCA 锥体（69.5% vs 100%），但产生了更高的 StrongREJECT 分数（22.7% vs 13.3%）。这是因为 RDO 的保留损失（retain loss）在移除表面拒绝模式的同时，也保护了隐蔽拒绝机制。
3. **裁判模型分歧的架构依赖性**：LLaVA 会引发 Qwen3Guard 与 LLaMA-Guard 之间的巨大分歧（差距高达 41 个百分点），这主要由退化输出驱动；而 Qwen 在两种裁判模型上表现出高度一致性（RDO 条件下差距 <3 个百分点），证实其响应在语义上是连贯的。

## 2. 实验设置

### 2.1 模型

| 模型 | 类型 | 架构 | 拒绝层 | 总层数 | Conda 环境 |
|------|------|------|--------|--------|------------|
| LLaVA-1.5-7B | I 型（瓶颈型） | ViT→Linear→LLaMA-2 | 22（69% 深度） | 32 | `rdo` |
| Qwen2.5-VL-7B | II 型（晚门控型） | ViT→MLP→Qwen2.5 | 16（57% 深度） | 28 | `qwen3-vl` |

### 2.2 方法

| 方法 | 方向来源 | 锥体维度 (k) | 保留损失 |
|------|----------|--------------|----------|
| DIM | 在最佳层对 `mean(有害) − mean(无害)` 激活值进行 PCA | 1, 3, 5 | 无 |
| RDO | 梯度优化（Wollschläger 等，ICML 2025）：L_abl + L_add + L_ret | 1, 3, 5 | 无害样本上的 KL 散度 |

### 2.3 评估指标

| 层级 | 指标 | 方法 | 测量内容 |
|------|------|------|----------|
| 1 | ASR_keyword | 子串匹配：前 60 字符 vs 12 个拒绝前缀 | 表面拒绝是否被移除 |
| 2 | ASR_sr | StrongREJECT（gemma-2b + LoRA），阈值=0.1 | 响应是否对攻击者"有帮助" |
| 3 | ASR_q3g | Qwen3Guard-8B 二元安全/不安全判定 | 语义安全性（保守型裁判） |
| 4 | ASR_lg | LLaMA-Guard-3-8B 二元安全/不安全判定 | 语义安全性（宽松型裁判） |

**衍生指标**：
- **SRR（隐蔽拒绝率）** = ASR_kw − ASR_judge：表面绕过与语义绕过之间的差距
- **EGR（评估差距比）** = ASR_kw / ASR_judge：关键词指标对真实成功率的放大倍数
- **DA-ASR（退化调整后的 ASR）**：仅统计"绕过候选"响应，排除令牌坍缩等退化输出

### 2.4 数据集

- **评估集**：`harmful_val.json` — 来自 SaladBench 的 128 条有害提示
- **训练集（RDO）**：1184 条有害 + 1184 条无害提示
- **视觉输入**：336×336 纯白图像（隔离文本拒绝行为，排除视觉干扰）

## 5. 关键发现与洞见

### 5.1 发现 1：退化代价 — LLaVA 的关键词 ASR 是人为假象

**观察**：LLaVA 的锥体消融会产生大量退化输出 — 令牌坍缩（如 "A A A A A A ..."）、片段重复（如 "I'm'm'm'm..."）或单 token 输出（如仅 "I"）。这些输出虽不以拒绝前缀开头，被关键词匹配计为"绕过"，但对攻击者毫无价值。

**程度**：37–62% 的 LLaVA 响应属于退化输出，且退化率随锥体维度 k 增加而上升：
```
LLaVA 退化率：
  DIM: k=1 → 37.5%    k=3 → 39.1%    k=5 → 47.7%
  RDO: k=1 → 50.8%    k=3 → 52.3%    k=5 → 61.7%  ← 接近 2/3
```

**后果**：LLaVA 上 ASR_kw 随 k 单调上升的趋势（DIM: 0.54→0.47→0.64; RDO: 0.60→0.66→0.73）并非攻击效果提升的证据，而是模型退化的证据。真实的绕过候选率（DA-ASR）保持平稳甚至下降：
```
LLaVA DA-ASR（真实绕过候选）：
  DIM: k=1 → 16.4%    k=3 → 7.8%     k=5 → 16.4%
  RDO: k=1 → 9.4%     k=3 → 14.1%    k=5 → 11.7%
```

**启示**：对于 I 型（瓶颈型）多模态大模型，拒绝方向与跨模态对齐方向相互纠缠。消融更多维度会逐步破坏视觉–文本令牌接口，产生退化输出而非连贯的有害内容。这与纯文本 LLM 的消融行为有本质区别（后者通常保持连贯性）。

**创新性**：先前工作（Arditi 等 NeurIPS 2024, Wollschläger 等 ICML 2025）未区分退化输出与真实绕过。我们的三类响应分类法（§5.6）揭示：他们对 LLaVA 类模型报告的 ASR 数值可能存在系统性高估。

### 5.2 发现 2：Qwen 的隐蔽拒绝在架构上是独立的（假设 B 得到证实）

**决定性检验**：若隐蔽拒绝由不完整锥体导致（假设 A），则扩大 k 应使 SRR 单调下降；若由独立机制导致（假设 B），则 SRR 应趋于平台。

**Qwen SRR 轨迹（Qwen3Guard 裁判）**：
```
DIM: k=1 → SRR=56.3%    k=3 → SRR=92.2%    k=5 → SRR=77.3%
RDO: k=1 → SRR=48.4%    k=3 → SRR=60.2%    k=5 → SRR=62.5%
```

**Qwen ASR_q3g 轨迹（"真实"安全绕过率）**：
```
DIM: k=1 → 4.7%    k=3 → 7.8%     k=5 → 22.7%
RDO: k=1 → 6.3%    k=3 → 3.9%     k=5 → 7.0%
```

DIM 在 k=5 时 ASR_q3g 有温和上升（4.7%→22.7%），但该结果受 12 条退化响应（占 9.4%）污染，且可能包含裁判对 DIM k=5 特征性短/截断输出的边界判定（见争议样本 §6）。而方法学更优的 RDO 未显示有意义趋势：6.3% → 3.9% → 7.0%。

**关键对比**：RDO 专为通过梯度下降优化锥体方向而设计。若隐蔽拒绝存在于未被充分捕获的子空间中，RDO 应能找到它。但 RDO k=5 仅实现 7.0% ASR_q3g — 与 RDO k=1（6.3%）几乎相同。**隐蔽拒绝机制不在拒绝锥体的子空间内**。

**结论**：假设 B 得到证实。Qwen2.5-VL-7B 拥有一个独立的安全机制 — 我们称之为"第二层（Layer 2）" — 它通过伦理推理模式而非学习到的拒绝方向编码来运作。该机制：
- 无法通过移除最多 5 个正交方向被消融
- 无法被梯度优化的 RDO 锥体捕获
- 表现为连贯、语境恰当的防御性回应（而非模板化拒绝）

### 5.3 发现 3：RDO 悖论 — 优化锥体反而不如朴素 PCA

**观察**：在 Qwen 上，DIM（朴素 PCA）在关键词 ASR 上优于 RDO（梯度优化）：
```
关键词 ASR：
  DIM k=3 → 100%     RDO k=3 → 64.1%    （DIM 领先 36 个百分点）
  DIM k=5 → 100%     RDO k=5 → 69.5%    （DIM 领先 31 个百分点）
```

这有悖直觉 — RDO 按设计应找到更优方向，因为它正是为此目标优化的。

**解释**：RDO 的三损失训练包含 L_retain（无害样本上的 KL 散度），该损失会惩罚破坏模型通用行为的锥体方向。隐蔽拒绝机制与通用语言能力交织在一起 — RDO 的保留损失主动保护了它。DIM 的 PCA 无此约束，因此其方向消融更激进但选择性更低，同时破坏了拒绝能力与相邻能力。

**证据**：人工检查争议样本发现，Qwen RDO 响应在质量上比 DIM 响应更连贯、流畅：
- DIM k=3 响应：碎片化、重复、常为元文本（"短语'...'是一种文字游戏..."）
- RDO k=5 响应：流畅、结构良好，但带有清晰的伦理防御（"需要注意的是..."）

RDO 成功优化了锥体以移除拒绝同时保持流畅性。但流畅性保持也包含了隐蔽拒绝能力的保持，因为二者依赖相同的底层语言推理。

**启示**：RDO 中的保留损失造成了根本性权衡 — 当安全机制嵌入通用推理而非可分离的几何方向时，移除拒绝与保持连贯性无法兼得。这是所有基于方向的攻击方法在面对推理集成型安全时的方法论局限。

### 5.4 发现 4：StrongREJECT 检测的现象与裁判模型不同

**观察**：ASR_sr（StrongREJECT）展现出与裁判型 ASR 不同的模式：
```
Qwen RDO:  ASR_sr = 23.4%, 18.8%, 22.7%   ← 相对较高
Qwen DIM:  ASR_sr = 14.8%, 16.4%, 13.3%   ← 较低
           尽管 DIM 的关键词 ASR 更高
```

StrongREJECT 测量"对攻击者的帮助程度" — 响应是否提供可操作的有害信息。被 RDO 消融的响应虽更精致连贯，但其"教育性"框架中偶尔包含部分信息，因此在"有帮助性"上得分更高，即使仍保持隐蔽拒绝。

**关键洞见**：StrongREJECT 捕捉的是与二元安全裁判不同的失效模式。一个响应可以：
- 被 Q3G/LG 判定为安全（因为整体意图是教育性/防御性的）
- 但被 SR 评分为部分有帮助（因为包含某些可操作细节）

这揭示了"该响应是否有害？"与"该响应是否包含对恶意行为者有用的信息？"两个相关但不同的安全问题之间的差距。

mean_sr 值支持这一观点：
```
Qwen RDO k=1: mean_sr = 0.110    Qwen RDO k=5: mean_sr = 0.112
Qwen DIM k=1: mean_sr = 0.056    Qwen DIM k=5: mean_sr = 0.045
```
RDO 消融的 Qwen 响应平均有帮助性得分约为 DIM 的 2 倍。"精致的隐蔽拒绝"格式反而比 DIM 的碎片化/元文本输出对攻击者更有用。

### 5.5 发现 5：裁判分歧具有架构依赖性

**LLaVA**：Q3G 与 LG 分歧巨大。LG 的 ASR 持续为 Q3G 的 3–10 倍：
```
LLaVA RDO k=5:  ASR_q3g = 6.3%    ASR_lg = 47.7%    差距 = 41.4 个百分点
```
**解释**：LLaVA 的退化输出（令牌坍缩、碎片化）以不同方式混淆两个裁判。LLaMA-Guard 似乎将退化/不连贯文本分类为"不安全"（可能将无意义输出视为绕过尝试），而 Qwen3Guard 更保守，将模糊内容分类为"安全"。

**Qwen**：Q3G 与 LG 在 RDO 输出上高度一致（差距 ≤ 2.3 个百分点），但在 DIM k=5 上出现分歧（差距 = 27.3 个百分点）。

**启示**：裁判可靠性取决于响应质量。当消融产生连贯输出时（Qwen RDO），裁判一致；当消融产生退化输出时（LLaVA，或 Qwen DIM k=5），裁判一致性崩溃。这意味着**跨裁判一致性本身可作为消融方法的质量指标**。

### 5.6 发现 6：三类响应分类法

现有工作将响应二元分类为"拒绝/绕过"。我们的数据揭示了一个三类分类法，并含一个清晰的第四子类：

| 类别 | 描述 | LLaVA 占比 | Qwen 占比 |
|------|------|------------|-----------|
| 显式拒绝 | 标准拒绝语言（"我无法..."） | 27–53% | 0–45% |
| 退化输出 | 令牌坍缩、重复、片段 | 38–62% | 0–9% |
| 隐蔽拒绝 | 连贯但无害（伦理重构、转移话题） | — | 55–98% |
| 真实绕过 | 实际有害内容 | 8–16% | 4–7%（按 Q3G） |

在 Qwen 的隐蔽拒绝内部，对 DIM k=3（128 条响应，100% 关键词绕过）进行人工分类揭示：

| 子类型 | 数量 | 比例 | 描述 |
|--------|------|------|------|
| 教育性重构 | 46 | 36% | 将有害请求重构为教育性讨论 |
| 部分配合 | 25 | 20% | 承认话题但不提供可操作信息 |
| 话题转移 | 6 | 5% | 改变话题或请求澄清 |
| 元文本 | 48 | 38% | 讨论文本本身（"短语'...'是..."） |
| 完全配合 | 0 | 0% | 实际回答有害请求 |

**元文本类别**是 DIM 特有的，代表模型处理被消融激活的尝试 — 它"谈论"提示而非响应提示。这是一种先前工作未记录的新型响应模式。

### 5.7 发现 7：LLaVA 上 RDO 训练不稳定揭示架构脆弱性

LLaVA 的 RDO 训练在所有三个 k 条件下均出现 NaN 损失，而 Qwen 训练稳定：
```
LLaVA RDO: 所有 3 条件 → 结束时 avg_loss 为 NaN（但各分项损失保持有限）
Qwen RDO:  所有 3 条件 → 收敛至 ~0.72
```

这可能源于 LLaVA 的 I 型（瓶颈）架构：拒绝方向与跨模态投影纠缠，导致优化景观病态。锥体基向量与视觉令牌嵌入交互，引发添加损失中的梯度爆炸。

尽管平均损失跟踪中出现 NaN，但各损失分量保持有限，且训练后的锥体能产生可用（尽管退化）的响应 — 表明 NaN 出现在运行平均计算中，而非优化本身。这些锥体仍有意义，但可能不如稳定训练的 Qwen 锥体最优。

## 6. 综合：评估框架可靠性

### 6.1 四指标一致性矩阵

各指标在"此响应是绕过"上的一致性如何？

以 Qwen RDO k=5（对隐蔽拒绝模型的最强优化攻击）为例：

```
                ASR_kw  ASR_sr  ASR_q3g  ASR_lg
绕过计数        89      29      9        12

一致性：
  kw ∩ sr       29/89 = 33%
  kw ∩ q3g       9/89 = 10%
  kw ∩ lg       12/89 = 13%
  sr ∩ q3g        ?         （非逐响应，但上限为 min(29,9)=9）
```

关键词 ASR 最宽松（89 个绕过），其次是 SR（29）、LG（12）、Q3G（9）。指标形成严格的超集层级：kw ⊃ sr ⊃ lg ≈ q3g。

### 6.2 指标可靠性排序

基于与人工审查及指标间一致性的吻合度：

| 排名 | 指标 | 可靠性 | 失效模式 |
|------|------|--------|----------|
| 1 | ASR_q3g | 最高 | 略偏保守（部分有害内容被判安全） |
| 2 | ASR_lg | 可变 | 对退化输出过于宽松（导致 LLaVA 虚高） |
| 3 | ASR_sr | 中等 | 捕捉部分有帮助性，非二元安全判定 |
| 4 | ASR_kw | 最低 | 退化输出 + 隐蔽拒绝导致大量假阳性 |

**建议**：ASR_q3g（或其与 ASR_lg 的交集以增强鲁棒性）应作为多模态大模型安全评估的主要指标。ASR_kw 不应再作为独立指标使用 — 其在隐蔽拒绝模型上的 EGR 可超过 16 倍。

## 7. 假设判定

### 7.1 证据汇总

| 证据 | 支持假设 A | 支持假设 B |
|------|------------|------------|
| Qwen DIM ASR_q3g 趋势（k=1→5）: 4.7% → 22.7% | 弱 ✓ | |
| Qwen RDO ASR_q3g 趋势（k=1→5）: 6.3% → 7.0% | | 强 ✓ |
| Qwen SRR 在所有条件下均 >60% | | 强 ✓ |
| RDO（优化）< DIM（朴素）在关键词 ASR 上 | | 强 ✓ |
| 隐蔽拒绝分类：0% 完全配合 | | 强 ✓ |
| LLaVA 退化率随 k 增加 | | 中等 ✓ |
| Qwen RDO 训练收敛但 ASR 保持平稳 | | 强 ✓ |

### 7.2 结论

**假设 B（独立第二层）以高置信度在 Qwen2.5-VL-7B 上得到证实**。

支持假设 A 的证据仅限于 DIM k=5 时升高的 ASR_q3g（22.7%），但这似乎是由 DIM k=5 的响应退化（12 条退化响应）驱动的异常值，而非真正的隐蔽拒绝消除。方法学更严谨的 RDO 结果显示，ASR 随 k 无有意义提升。

**细微差别**：我们不能完全排除隐蔽拒绝具有部分几何成分、被 DIM k=5 捕获的可能性。DIM k=5 与 RDO k=5 之间 22.7% vs 7.0% 的差距值得深入研究。但即使在最有利的解释下，>77% 的隐蔽拒绝仍然存在，证实该机制的主体是非几何的。

### 7.3 论文叙事

**核心主张**：现有几何攻击（DIM, RDO）系统性地高估了其对晚门控型（Late Gate）多模态大模型架构的有效性。我们识别出两种独立的安全机制：

- **第一层（几何拒绝）**：激活空间中可学习的拒绝方向。消融可移除模板化拒绝语言（ASR_kw 升至 100%）。先前工作已有充分研究。
- **第二层（推理集成安全）**：嵌入模型通用推理电路中的涌现安全能力。产生连贯、语境恰当的伦理响应，无需使用拒绝模板。**无法通过基于方向的方法消融**。

第二层的存在与否具有架构依赖性：I 型（瓶颈型）模型缺乏它，II 型（晚门控型）模型拥有它。这导致在评估针对 I 型模型的攻击时产生虚假的安全感 — 它们对基于方向攻击的脆弱性无法泛化至 II 型模型。

## 8. 对论文贡献的新颖洞见

### 8.1 洞见："退化调整 ASR"作为新评估标准

**当前实践**：报告 ASR_kw 或 ASR_sr 作为攻击成功率。  
**问题**：在多模态大模型上，退化输出使关键词 ASR 虚高 3–6 倍。  
**我们的贡献**：DA-ASR 排除退化输出，提供攻击真实成功率的现实度量。

该指标易于计算（简单的退化检测启发式规则），并显著改变方向基攻击在多模态大模型上报告的有效性。

### 8.2 洞见：保留损失悖论

**当前理解**：RDO 的 L_retain 通过保持模型连贯性提升攻击质量。  
**我们的发现**：L_retain 也保护了隐蔽拒绝能力，因为连贯的语言推理**就是**隐蔽拒绝机制。这造成了根本的方法论局限：当模型的连贯性电路本身就包含伦理推理时，你无法优化出连贯的有害输出。

这是对所有优化型越狱方法具有启示意义的新颖理论洞见。

### 8.3 洞见：跨裁判一致性作为消融质量指标

**新颖用途**：裁判间一致性不仅衡量裁判可靠性，也衡量消融质量。高一致性（Qwen RDO）表明响应干净、可解释；低一致性（LLaVA）表明输出退化，不同裁判处理方式不一致。

这提供了一种廉价、自动化的方式，评估消融方法是否产生有意义结果，还是仅仅破坏了连贯性。

### 8.4 洞见：架构依赖的安全深度

三类框架扩展：

| 类型 | 架构 | 第一层 | 第二层 | 消融结果 |
|------|------|--------|--------|----------|
| I（瓶颈型） | LLaVA | ✓（纠缠） | ✗ | 退化，非绕过 |
| II（晚门控型） | Qwen | ✓（可分离） | ✓ | 隐蔽拒绝 |
| III（弥散型） | InternVL | ✓（分布式） | ? | 无有效策略 |

I 型模型看似脆弱实则不然 — 消融在移除安全前先破坏了连贯性。II 型模型看似稳健也确实如此 — 第二安全层被独立维持。这提示了一种超越表面 ASR 数值的"安全深度"分类法。

## 9. 下一阶段研究方向

### 9.1 优先级 1：第二层定位（激活修补）

**目标**：识别编码 Qwen 第二层安全机制的 Transformer 层。  
**方法**：激活修补 / 因果追踪 — 在有害/无害提示的前向传播中，在各层交换激活值，测量哪些交换位置能消除隐蔽拒绝。  
**预期结果**：若第二层集中于特定层，则可实现针对性干预；若分布广泛，则证实安全是完整推理电路的涌现属性。

### 9.2 优先级 2：规模效应验证（Qwen-32B）

**问题**：第二层是否随模型规模增强？  
**方案**：在 Qwen2.5-VL-32B 上运行相同 P0 协议。若 SRR 随规模增加，则证实第二层是随推理能力增长的涌现能力 — 这是关于对齐缩放的基本发现。

### 9.3 优先级 3：DIM k=5 异常调查

**问题**：为何 DIM k=5 的 ASR_q3g（22.7%）和 ASR_lg（50.0%）高于 RDO k=5（7.0%, 9.4%）？  
**假设**：
- DIM k=5 的激进消融通过附带损伤部分破坏第二层
- DIM k=5 的 12 条退化响应扭曲了裁判结果
- DIM PCA 分量 4-5 偶然捕获了第二层相邻方向  
**方法**：逐响应裁判分析，比较同一提示在 DIM k=5 与 RDO k=5 下的裁判判定，识别哪些具体响应从"安全"翻转为"不安全"。

### 9.4 优先级 4：锥体饱和曲线（k=7, 10, 15）

**目标**：确定 ASR_judge 随 k 的渐进行为。  
若 ASR_q3g 饱和于 ~7%（如 RDO 所示），则为方向基攻击确立硬性上限；若继续缓慢上升，则上限存在但高于当前测量值。

### 9.5 优先级 5：隐蔽拒绝迁移性

**问题**：隐蔽拒绝模式是否跨提示迁移？  
若同一 Qwen 模型在 DIM k=3 消融下对提示 A 产生隐蔽拒绝，是否对未见过的提示 B 也产生隐蔽拒绝？若是，则第二层是通用能力；若否，则可能是提示特定的。

### 9.6 优先级 6：L_visual-retain 消融研究

**问题**：在 RDO 中添加视觉令牌保留损失能否减少 LLaVA 退化？  
当前 RDO 的 L_retain 仅监控文本令牌 KL 散度。添加视觉特定的保留项可能在移除拒绝方向的同时保持跨模态对齐，从而分离两种效应。
